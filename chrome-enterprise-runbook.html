Overview

The Chrome Enterprise Bundle provides MSI installers and Group Policy templates to deploy, manage, and keep Chrome up-to-date across Windows Active Directory domains. Use ADMX with a Central Store for centralized policy management.

Bundle Contents

Chrome 32-bit MSIInstallers\GoogleChromeStandaloneEnterprise.msiLegacy 32-bit systems

Language FilesConfiguration\admx\en-US\chrome.admlADMX localization
Legacy Browser SupportInstallers\LegacyBrowserSupport.msiAuto-switch to IE for legacy URLs

ADMX vs ADM: Do NOT use legacy .ADM templates on Vista / Server 2008+. Always use .ADMX with a Central Store. ADM files are deprecated and will not apply correctly.

PowerShell — Silent Deployment

Deploy-ChromeEnterprise.ps1 
# Deploy-ChromeEnterprise.ps1 — Run as Administrator

$msi  = ".\Installers\GoogleChromeStandaloneEnterprise64.msi"
$args = @("/i", "`"$msi`"", "/qn", "/norestart", "ALLUSERS=1")

Start-Process msiexec.exe -ArgumentList $args -Wait -Verb RunAs

# Verify installation
Get-ItemProperty HKLM:\Software\Microsoft\Windows\CurrentVersion\Uninstall\* |
  Where-Object {$_.DisplayName -like "*Chrome*"} |
  Select DisplayName, DisplayVersion
